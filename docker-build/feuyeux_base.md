Docker-Ubuntu14.04 基础镜像
----
####0 创建Docker工作目录
```
[erichan@mars-centos7 ~]$ mkdir docker-room
[erichan@mars-centos7 ~]$ cd docker-room/
[erichan@mars-centos7 docker-room]$ mkdir ubuntu-base
```

####1 制作镜像
```
[erichan@mars-centos7 docker-room]$ nano Dockerfile
# Version: 0.0.1
FROM ubuntu:14.04
MAINTAINER Eric Han "feuyeux@gmail.com"
RUN apt-get update
RUN apt-get install -y openssh-server
RUN mkdir /var/run/sshd
RUN echo "root:password" | chpasswd
EXPOSE 22

```

####2 构建镜像
```
[erichan@mars-centos7 docker-room]$ sudo docker build -t="feuyeux/ubuntu-base" .
```
